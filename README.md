## 🎓 Accredian Enterprise Clone

A modern, responsive **enterprise training platform UI clone** enhanced with an **AI-powered chatbot (voice-enabled)**.  
Built using **Next.js, Tailwind CSS, and Gemini API**, this project focuses on real-world UI replication, responsiveness, and intelligent user interaction.

---

## 🎯 Project Overview

This project replicates the **Accredian Enterprise website** with a strong focus on:

- Pixel-perfect UI implementation  
- Fully responsive design  
- AI-powered chatbot integration  
- Voice interaction (speech-to-text + text-to-speech)  

It combines **frontend engineering + AI integration** to deliver a smart and interactive user experience.

## 🚀 Key Features Implemented

### 🎨 UI & UX
- Pixel-perfect UI matching original Accredian website  
- Fully responsive (mobile, tablet, desktop)  
- Clean modern layout with Tailwind CSS  

---

### 🤖 AI Chatbot
- AI-powered chatbot using Gemini API  
- Context-aware responses for training queries  
- Floating chatbot UI  

---

### 🎤 Voice Features
- Voice input (Speech Recognition)  
- Voice output (Text-to-Speech)  
- Hands-free interaction  

---

### 📩 Contact System
- Interactive enquiry modal  
- Structured form UI  
- Ready for backend integration  

---

### ⚡ Performance
- Optimized components  
- Clean state management  
- Fast rendering with Next.js  

---

### 🔐 Security
- API key secured via backend route  
- Environment variable handling
